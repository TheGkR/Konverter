# Konverter
A Web base Converter with Converters Such As:

#### CSV <---> JSON
#### Markdown ---> HTML
#### Text --->Base64
#### Image <---> Base64 (The Base64 to image can crash the webpage :/ if the image size is too big)
#### Hex <---> RGB
#### Hash Generator(SHA256 & MD5)
#### Image Format Converter (PNG, JPG, GIF, BMP, TIFF, WebP)
